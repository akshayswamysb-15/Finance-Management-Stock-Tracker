# Personal Finance Manager

A comprehensive desktop application built with Python and wxPython to help users manage their personal finances. This app combines tax calculation, stock tracking, and account management in a single, user-friendly interface.

## Features

- Login/Register/Forgot Password: Secure user authentication with a security question for password recovery.
- Dashboard: Provides an overview and easy navigation to all features.
- Tax Calculator: Calculate taxes under Old and New regimes, with optional deductions.
- Tax History: View past tax calculations and export them as PDF reports.
- Stock Tracker: Monitor simulated stock prices and visualize trends.
- Watchlist: Add favorite stocks to a personal watchlist and track them.
- Account Management: Update username/password or delete account securely.
- Logout: Safely log out of the application.

## Technologies Used

- Python 3
- wxPython for GUI
- Matplotlib for stock trend visualization
- ReportLab for generating PDF reports
- CSV for persistent data storage
